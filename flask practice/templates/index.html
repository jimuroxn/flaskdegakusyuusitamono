ネコの紹介


マッシュ

飼い猫です　彼の特徴は運動神経が良いことです　いつもタンスの上にもよく登ります　 彼は甘えん坊で、よくすり寄ってきたり、家族の近くで自分の肉球をちゅぱちゅぱすっています　 これは彼が十分に親猫と一緒にいることが出来なかったから、人間を親代わりにしているのかもしれません　

のび太

飼い猫です　彼の特徴は運動神経が悪いことです　よくマッシュと場所の取り合いをして、負けています　 またドジでもあり、高いところで寝ているとたまに落ちそうになっています　 マッシュほど甘えん坊ではなく、せいぜいが頭をこすりつけてくるくらいです　 免疫が弱いのか、よく目が赤くなっているのを見ます　そういう時は大体おとなしいので、痛いのかもしれません　

苦労したこと

まず、仮想環境の構築に手間取った。Anacondaで Python を始めたため、Anacondaprompt か AnacondaNavigator を使って環境を構築しなければならないと気づくのに時間がかかった。
また、Flaskの url_for で Bootstrap を使おうとしたけれどなぜかうまくいかず、違う手段で読み込むことにしたのは良かったのかわからない。
他にも、失敗したこと、解決したことはいくつかあるけれど、全て自分で調べて解決した。自分で調べて解決し、形のあるものを作るというのは、時間はかかるけれども学習にはとても良いと 実感した。（少し時間がかかりすぎたような気はする）
